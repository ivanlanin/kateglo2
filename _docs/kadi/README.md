# Kadi — Kamus Deskriptif Indonesia

Subproyek dari **Kateglo 2.0**. Kadi adalah sistem leksikografi deskriptif berbasis
atestasi yang menjaring, mengklasifikasikan, dan mengkurasi neologisme bahasa Indonesia
secara berkelanjutan dari korpus multi-genre.
