# Inventaris Heading Internal Halaman Gramatika

Tanggal: 2026-03-23

Dokumen ini merekam semua file `frontend/public/gramatika` yang masih memiliki heading internal `##`, `###`, dan seterusnya. Untuk setiap heading dicantumkan nomor baris sumber dan kandidat slug awal yang bisa dipakai saat file dipecah.

Catatan:

- `[BENTROK]` berarti kandidat slug bertabrakan dengan slug/file yang sudah ada.
- Kandidat yang diawali angka atau kehilangan informasi fonetik perlu dinormalisasi lagi saat implementasi.
