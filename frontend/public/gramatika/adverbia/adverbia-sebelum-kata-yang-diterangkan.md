---
id: 6.3.1
title: Adverbia Sebelum Kata yang Diterangkan (6.3.1)
---

Dalam struktur kalimat, adverbia ini digunakan sebelum, atau mendahului, kata yang diterangkan.

Contoh:

(19)

| Adverbia | + | Kata yang diterangkan |
|----------|---|----------------------|
| *baru* | | *tiba* |
| *hanya* | | *sedikit* |
| *jarang* | | *datang* |
| *lebih* | | *tinggi* |
| *selalu* | | *hadir* |
| *paling* | | *rajin* |
| *sering* | | *mogok* |
| *sangat* | | *pandai* |
| *terlalu* | | *sulit* |

(20)
- a. Ia *lebih* tinggi daripada adiknya.
- b. Telaga itu *sangat* indah.
- c. Pendiriannya *terlalu* kukuh untuk digoyahkan.
- d. Kami *hanya* menulis apa yang dikatakannya.
- e. Murid yang *paling* pintar mendapat hadiah dari sekolahnya.
