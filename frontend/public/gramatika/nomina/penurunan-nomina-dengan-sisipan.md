id: 7.1.4.2.2.2.8
title: Penurunan Nomina dengan Sisipan
---

Penurunan nomina dengan memakai sisipan atau infiks dalam bahasa Indonesia tidak produktif lagi. Bentuk-bentuk turunan dengan infiks yang ada sudah membeku.

> *telunjuk* <- *tunjuk*; *pelatuk* <- *patuk*; *gelembung* <- *gembung*; *telapak* <- *tapak*; *kelelawar* <- *kelawar*; *kemilau* <- *kilau*; *kemuning* <- *kuning*; *temali* <- *tali*; *serabut* <- *sabut*; *seruling* <- *suling*; *gerigi* <- *gigi*; *kinasih* <- *kasih*; *sinambung* <- *sambung*; *tinambah* <- *tambah*
